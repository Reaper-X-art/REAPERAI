Network Sniffer

This project is a network sniffer written in C that can capture and analyze raw Ethernet traffic. It records all incoming and outgoing packets when a device is connected through an adapter and cable. The sniffer forwards the gathered information to a secure server for storage and inspection. Its operation is silent in the background, allowing detailed network monitoring without disrupting the target system.
